Welcome

This project measures grocery inflation using real in-store prices for a consistent set of essential products. Our primary output is a monthly average for each item, updated from weekly observations.

Method (Process)

  • Collection frequency: prices are recorded once per week.
  • Unit price focus: the project compares item prices as consistently as possible across stores.
  • Product consistency: items are recorded using consistent quality, size, and type whenever possible.
  • Loyalty pricing: loyalty-card-only prices are ignored for official averages.
  • Promotions: sale prices may be noted for context.
  • Substitutions: if an exact item is unavailable, the closest valid equivalent is recorded and documented.
  • Reporting: the main output is a monthly average per item, built from weekly observations.
  • Continuation: this project is designed so future groups can continue collecting and comparing data.
