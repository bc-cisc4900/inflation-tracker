Welcome

This project measures grocery inflation using real in-store prices for a consistent set of esential products.
Our primary output is a monthly average for each item, updated from weekly observations.

Method (Process)

  • Collection frequency: prices are recorded once per week.
  • Unit price focus: we compare unit price to reduce packaging bias.
  • Product consistency: items are recorded using consistent quality (e.g., “large white eggs”). Equivalent generic products may be used.
  • Loyalty pricing: loyalty-card-only prices are ignored for official averages.
  • Promotions: sales/promotions may be recorded as metadata for context.
  • Anonymization: store and brand identifiers are hidden (e.g., “Store A”, “Brand X”).
  • Reporting: the main output is a monthly average per item (built from weekly observations).
  • Continuation: this project is designed to be continued by future cohorts to complete the remaining months.
