# Technology Stack

## Frontend
- HTML for structure
- CSS for styling
- JavaScript for interactivity

## Backend / Import Tools
- Node.js
- JavaScript
- npm packages defined in `grocery-importer/package.json`

## Database / Data
- MariaDB for relational data storage
- SQL for schema design, setup, and query logic
- Google Sheets for weekly grocery data collection

## Documentation / Workflow
- Git and GitHub for version control
- GitHub Issues and Pull Requests for team workflow
- Brooklyn College UNIX / web server environment for deployment and database access
