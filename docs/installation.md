# Installation Instructions

## Clone the Repository
Clone the GitHub repository to your local machine and enter the project folder.

## Website Setup
1. Open the `website/` folder in a code editor.
2. Edit `Web.html` or other website files as needed.
3. Upload website files to the Brooklyn College web server for hosting.

## Grocery Importer Setup
1. Open the `grocery-importer/` folder.
2. Install required Node.js packages using:
   `npm install`
3. Review `package.json` for dependencies and scripts.

## Database Setup
1. Log into a Brooklyn College UNIX account.
2. Open the MariaDB client.
3. Run SQL files from the `database/` folder as needed.

## Local Development
The website can be previewed locally by opening `website/Web.html` or `website/index.html` in a web browser.
