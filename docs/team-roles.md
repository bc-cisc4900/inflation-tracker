## Role Notes
Because the group is small, responsibilities may overlap. Team members may contribute across documentation, testing, data collection, planning, and technical implementation as needed.

## Responsibility Summary

### Project Manager / Database Architect
Responsible for coordinating project direction, organizing requirements, contributing to documentation, and helping define database structure and data workflow.

### Backend Developer
Responsible for backend-related work such as API planning, server-side logic, importer/backend tooling, and database interaction support.

### Frontend Developer
Responsible for website layout, presentation, structure, and display of project data on the frontend.

### Data Collection / QA
Responsible for collecting grocery price data, helping verify consistency of items and substitutions, and checking that the information being recorded is usable and clear.

## Shared Expectations
All team members are expected to:
- communicate progress
- document important work
- help maintain project organization
- support live demo and presentation preparation
- contribute to the overall success of the prototype
